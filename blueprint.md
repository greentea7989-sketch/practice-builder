# Blueprint: 메세지 관계론 (Message-Relationship Theory)

## 1. 앱 개요 (Overview)

\'메세지 관계론\'은 사용자가 상대방과의 관계 및 성격적 특성을 고려하여, 더욱 효과적이고 세련된 메시지를 작성할 수 있도록 돕는 웹 애플리케이션입니다. 사용자가 전달하고자 하는 핵심 내용을 입력하면, 앱이 대화의 맥락과 톤을 분석하여 최적화된 여러 버전의 메시지를 추천합니다.

## 2. 핵심 기능 및 디자인 (Features & Design)

### 스타일 및 디자인 (Style & Design)
- **UI:** 모던하고 깔끔한 인터페이스를 중심으로 구성하며, 사용자가 쉽게 상호작용할 수 있도록 직관적인 UI/UX를 제공합니다.
- **Color Palette:** 신뢰감을 주는 네이비와 블루 계열을 메인 색상으로 사용하고, 포인트 색상으로 생동감 있는 코랄을 활용하여 활기찬 느낌을 더합니다.
- **Typography:** 가독성이 높은 \'Pretendard\' 폰트를 사용하여 정보 전달력을 높입니다.
- **Layout:** 카드형 UI를 사용하여 각 기능 영역(입력, 분석 결과, 추천 메시지)을 명확하게 구분합니다. 부드러운 그림자 효과와 미묘한 배경 텍스처를 추가하여 깊이감과 고급스러운 질감을 표현합니다.
- **Interactivity:** 버튼, 입력창 등 상호작용 요소에는 호버 및 클릭 시 섬세한 애니메이션과 \'glow\' 효과를 적용하여 사용 경험을 향상시킵니다.

### 기능 구현 (Features)
- **관계 정의:** 사용자와 상대방의 관계를 텍스트로 입력받습니다. (예: "동료", "친구", "연인")
- **성격 분석:**
    - **메시지 스크린샷 업로드:** 주고받은 대화 스크린샷을 최대 3개까지 업로드하여 대화 패턴, 반응, 자주 사용하는 어휘 등을 분석합니다.
    - **관계 정의 기반 추론:** 입력된 관계를 기반으로 일반적인 소통 방식을 추론합니다.
- **메시지 코칭:**
    - **원본 메시지 입력:** 사용자가 보내고 싶은 메시지의 초안을 입력하는 텍스트 영역을 제공합니다.
    - **스타일 톤 조절:** \'진지하게\', \'유머러스하게\', \'친근하게\' 등 원하는 메시지 톤을 선택할 수 있는 버튼을 제공합니다.
    - **코칭 결과 생성:** 위의 정보들을 종합하여, 다양한 버전의 추천 메시지를 생성하여 보여줍니다.
- **첫 멘트 생성:** 간단한 키워드나 상황을 입력하면 대화를 시작할 수 있는 오프닝 멘트를 자동으로 생성해주는 기능을 포함합니다.
